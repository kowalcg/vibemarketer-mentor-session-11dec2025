## ðŸ“± App Store Tools Setup

### Apple Developer Account

1. **Visit:** [developer.apple.com](https://developer.apple.com)
2. **Sign In:** Apple ID
3. **Enroll:** $99/year
4. **Complete:**
   - Personal information
   - Banking (if monetizing)
   - Tax information

### App Store Connect

1. **Visit:** [appstoreconnect.apple.com](https://appstoreconnect.apple.com)
2. **Sign In:** Apple Developer account
3. **Accept Agreements:**
   - Paid Applications Agreement
   - Free Applications Agreement
4. **Create First App:**
   - Bundle ID
   - App name
   - Primary language

### Xcode Setup (If Using Manual Submission)

1. **Download:** From Mac App Store
2. **Install:** Large download, be patient
3. **Configure:**
   - Sign in with Apple ID
   - Accept licenses
   - Install additional components

### Expo Setup

1. **Install Node.js:** [nodejs.org](https://nodejs.org)
2. **Install Expo CLI:**
   ```bash
   npm install -g expo-cli
   ```
3. **Install Expo Go App:** On your iPhone
4. **Test:**
   ```bash
   expo init MyApp
   cd MyApp
   expo start
   ```

**Resource:** [Expo Documentation](https://docs.expo.dev/)

## ðŸŽ¤ Dictation Tools Setup

### ChatGPT Voice (Recommended)

1. **Open ChatGPT App:** iOS/Android
2. **Enable Voice:**
   - Settings â†’ Voice
   - Choose voice
   - Test dictation
3. **Use:**
   - Tap microphone icon
   - Speak up to 10 minutes
   - AI processes automatically

**Pro Tip:** Explain prompts thoroughly. More context = better results.

### VibeCode Built-in Dictation

1. **Open VibeCode App**
2. **Find Dictation Button:** In prompt area
3. **Enable Permissions:** Microphone access
4. **Test:** Dictate a prompt

**Advantage:** No context switching, works directly in app.

### Apple Dictation (Alternative)

1. **System Settings:** macOS/iOS
2. **Enable Dictation:**
   - System Preferences â†’ Keyboard â†’ Dictation
   - Enable dictation
   - Choose shortcut
3. **Use:** Press shortcut, speak, press again

**Advantage:** Free, system-wide, always available.
