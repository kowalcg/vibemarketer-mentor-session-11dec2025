## Dashboard Overview

### Core Concept
A single-page HTML dashboard that serves as both:
- **Visual presentation aid** (projected on screen)
- **Navigation tool** (speaker controls flow)
- **Story delivery system** (guides narrative)
- **Interactive experience** (audience engagement)

### Design Philosophy
- **Dark mode** - Professional, easy on eyes during presentation
- **Card-based layout** - Each story beat in its own container
- **Story-driven flow** - Natural narrative progression
- **Visual anchors** - Screenshots, timelines, charts guide attention
- **Time indicators** - Help maintain 90-minute pacing
- **Smooth scrolling** - Seamless transitions between sections
- **Speaker notes** - Hidden overlay for talking points
